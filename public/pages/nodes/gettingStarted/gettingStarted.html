First, create a directory or a folder, then through terminal go to inside that directory and create app.js and package.json files. After that initiate all the require dependencies, in this case specifically initiate express dependencies, and set type to module as we may need to import and export files. Finally, execute "npm install" command.

After intalled all the required dependencies like express, Now import express in app.js file and set the port.

Now, create mapping and run the app.js file with node. We can run app.js file using either 'node' or 'nodemon.cmd'. The diffrence is with 'nodemon.cmd', the file restart automatically after each changes in files like html, css, js, etc. It reduces the extra work of run the program again and again.

Now, go to 'http://localhost:8080/'
